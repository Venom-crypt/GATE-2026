## DISCRETE MATHEMATICS

### Part 1: Propositional Logic

**Propositions:** Statements that are true or false (not both)

**Operators:**
- **NOT (¬):** Negation, opposite truth value
- **AND (∧):** Both must be true
- **OR (∨):** At least one true
- **XOR (⊕):** Exactly one true
- **IMPLIES (→):** p→q = ¬p ∨ q (false only if p true, q false)
- **IFF (↔):** p↔q = (p→q) ∧ (q→p) (true if same value)

**Truth Tables:** Complete specification of operator behavior

**Tautology:** Always true
**Contradiction:** Always false
**Contingency:** Sometimes true, sometimes false

### Part 2: First-Order Logic

**Predicates:** Functions with true/false values

**Quantifiers:**
- **∀ (Universal):** For all x, P(x)
- **∃ (Existential):** There exists x such that P(x)

**Rules:**
- ¬(∀x P(x)) ≡ ∃x ¬P(x)
- ¬(∃x P(x)) ≡ ∀x ¬P(x)

### Part 3: Sets, Relations, Functions

**Sets:** Collections of distinct objects
- **Union (∪):** All elements in either set
- **Intersection (∩):** Elements in both sets
- **Complement (A'):** Elements not in A
- **Cardinality |A|:** Number of elements

**Relations:** Subset of Cartesian product A × B

**Types:**
- **Reflexive:** (a,a) for all a
- **Symmetric:** (a,b) implies (b,a)
- **Transitive:** (a,b) and (b,c) implies (a,c)
- **Equivalence:** Reflexive, symmetric, transitive

**Functions:**
- **Domain:** Input set
- **Codomain:** Output set
- **Range:** Actual outputs
- **Injective (One-to-One):** Different inputs → different outputs
- **Surjective (Onto):** Every codomain element has preimage
- **Bijective:** Both injective and surjective

### Part 4: Partial Orders and Lattices

**Partial Order:** Reflexive, antisymmetric, transitive relation

**Lattice:** Partial order where every two elements have:
- **Greatest Lower Bound (GLB):** Meet
- **Least Upper Bound (LUB):** Join

### Part 5: Monoids and Groups

**Monoid:** Set with associative operation and identity element
- Example: Integers under addition with 0

**Group:** Monoid where every element has inverse
- **Abelian:** Commutative group
- Example: Integers under addition

**Cyclic Group:** Generated by single element

## GRAPH THEORY

### Part 1: Connectivity

**Graph:** Vertices (nodes) and edges connecting them

**Path:** Sequence of edges connecting vertices
**Cycle:** Path starting and ending at same vertex

**Connected:** Path exists between all pairs of vertices
**Component:** Maximal connected subgraph

**Bridge:** Edge whose removal disconnects graph
**Articulation Point:** Vertex whose removal disconnects graph

### Part 2: Matching and Coloring

**Matching:** Set of edges with no common vertices
**Maximum Matching:** Matching with most edges

**Graph Coloring:** Assign colors to vertices such that adjacent vertices have different colors
**Chromatic Number:** Minimum colors needed

**Bipartite Graph:** Vertices in 2 disjoint sets, edges between sets
**König's Theorem:** Maximum matching = minimum vertex cover in bipartite graphs

## COMBINATORICS

### Part 1: Counting

**Permutations:** P(n,r) = n!/(n-r)! (order matters)
**Combinations:** C(n,r) = n!/(r!(n-r)!) (order doesn't matter)

**Pigeonhole Principle:** If n+1 items in n containers, at least one has 2+

**Inclusion-Exclusion:** |A∪B| = |A| + |B| - |A∩B|

### Part 2: Recurrence Relations

**Recurrence:** Recursive formula for sequence
Example: a_n = a_{n-1} + a_{n-2} (Fibonacci)

**Solving Methods:**
1. Homogeneous: a_n = r^n
2. Particular solution for non-homogeneous
3. General solution: homogeneous + particular

### Part 3: Generating Functions

**Generating Function:** G(x) = Σ a_n x^n

**Use:** Encode sequence, solve recurrences, count structures

## LINEAR ALGEBRA

### Part 1: Matrices

**Matrix:** m×n array of numbers

**Operations:**
- **Addition:** Element-wise
- **Multiplication:** (m×n)(n×p) = m×p
- **Transpose:** Swap rows and columns

**Types:**
- **Symmetric:** A = A^T
- **Diagonal:** Non-zero only on diagonal
- **Identity:** I_n = diagonal with 1's

### Part 2: Determinants

**Definition:** Scalar associated with square matrix
- **2×2:** det = ad - bc
- **3×3:** Rule of Sarrus or cofactor expansion

**Properties:**
- det(AB) = det(A)det(B)
- det(A^T) = det(A)
- det(kA) = k^n det(A)
- Swapping rows changes sign

**Invertibility:** A invertible iff det(A) ≠ 0

### Part 3: System of Linear Equations

**Forms:**
- Ax = b (matrix form)
- x = A^(-1)b (solution if A invertible)

**Methods:**
- **Gaussian Elimination:** Row reduction
- **Cramer's Rule:** x_i = det(A_i)/det(A)

**Solutions:**
- **Unique:** det(A) ≠ 0
- **Infinite:** Rank(A) = Rank(A|b) < n
- **None:** Rank(A) < Rank(A|b)

### Part 4: Eigenvalues and Eigenvectors

**Eigenvalue λ:** Av = λv
**Eigenvector v:** Non-zero vector satisfying Av = λv

**Finding:**
1. Solve det(A - λI) = 0 (characteristic equation)
2. For each λ, solve (A - λI)v = 0

**Diagonalization:** A = PDP^(-1) where D diagonal of eigenvalues, P columns are eigenvectors

### Part 5: LU Decomposition

**Factorization:** A = LU where L lower triangular, U upper triangular

**Use:** Efficient solving Ax = b by solving Ly = b, then Ux = y

## CALCULUS

### Part 1: Limits and Continuity

**Limit:** L(x) approaches c as x→a
**Definition:** For all ε>0, ∃δ>0 such that |x-a|<δ ⟹ |f(x)-L|<ε

**Continuity:** f continuous at a if lim(x→a) f(x) = f(a)

**Types of Discontinuity:**
- **Removable:** Limit exists but not equal to function value
- **Jump:** Left and right limits exist but differ
- **Infinite:** Vertical asymptote

### Part 2: Differentiability

**Derivative:** f'(x) = lim(h→0) [f(x+h)-f(x)]/h

**Rules:**
- **Sum:** (f+g)' = f' + g'
- **Product:** (fg)' = f'g + fg'
- **Quotient:** (f/g)' = (f'g - fg')/g²
- **Chain:** (f∘g)' = f'(g(x))·g'(x)

**Common Derivatives:**
- (x^n)' = nx^(n-1)
- (e^x)' = e^x
- (ln x)' = 1/x
- (sin x)' = cos x

### Part 3: Maxima and Minima

**Critical Points:** f'(x) = 0 or undefined

**Second Derivative Test:**
- f''(x) < 0: Local maximum
- f''(x) > 0: Local minimum
- f''(x) = 0: Inconclusive

**Absolute Extrema:** Check critical points and endpoints

### Part 4: Mean Value Theorem

**MVT:** If f continuous [a,b] and differentiable (a,b), then ∃c∈(a,b): f'(c) = [f(b)-f(a)]/(b-a)

### Part 5: Integration

**Indefinite Integral:** ∫f(x)dx = F(x) + C where F'(x) = f(x)

**Definite Integral:** ∫[a,b] f(x)dx = F(b) - F(a)

**Rules:**
- **Sum:** ∫(f+g) = ∫f + ∫g
- **Substitution:** ∫f(g(x))g'(x)dx = ∫f(u)du
- **By Parts:** ∫u dv = uv - ∫v du

## PROBABILITY AND STATISTICS

### Part 1: Random Variables and Distributions

**Random Variable:** Function from outcomes to real numbers

**Probability Distributions:**

**1. Uniform:** U(a,b) - Equally likely on [a,b]
- PDF: f(x) = 1/(b-a)
- Mean: (a+b)/2
- Variance: (b-a)²/12

**2. Normal:** N(μ,σ²) - Bell curve
- PDF: f(x) = (1/(σ√(2π))) exp(-(x-μ)²/(2σ²))
- Mean: μ
- Variance: σ²
- 68-95-99.7 Rule

**3. Exponential:** λ - Memoryless
- PDF: f(x) = λe^(-λx)
- Mean: 1/λ
- Variance: 1/λ²

**4. Poisson:** λ - Count of events
- PMF: P(X=k) = (λ^k e^(-λ))/k!
- Mean: λ
- Variance: λ

**5. Binomial:** B(n,p) - Number of successes
- PMF: P(X=k) = C(n,k)p^k(1-p)^(n-k)
- Mean: np
- Variance: np(1-p)

### Part 2: Statistics

**Mean (μ):** Average value
**Median:** Middle value
**Mode:** Most frequent value

**Variance (σ²):** Average squared deviation from mean
σ² = Σ(x_i - μ)²/n

**Standard Deviation (σ):** √variance

**Coefficient of Variation:** σ/μ (relative variability)

### Part 3: Conditional Probability and Bayes' Theorem

**Conditional Probability:** P(A|B) = P(A∩B)/P(B)

**Bayes' Theorem:** P(A|B) = P(B|A)P(A)/P(B)

**Total Probability:** P(B) = Σ P(B|A_i)P(A_i)
